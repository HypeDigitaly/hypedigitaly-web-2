# HypeDigitaly Landing Page

A high-converting B2B SaaS landing page for AI transformation services.

## Project Structure

```
hypedigitaly-web-2/
├── index.html          # Main landing page
├── assets/
│   ├── images/
│   │   ├── clients/    # Client logos (Czech regions)
│   │   ├── logo.png    # HypeDigitaly logo
│   │   ├── pavel.jpg   # CEO photo
│   │   ├── favicon.ico
│   │   └── voiceflow-badge.png
│   └── videos/
│       └── krakonos.mp4  # Demo video
└── README.md
```

## Features

- **Dark Mode Design**: Futuristic dark theme with teal accents
- **Scroll Animations**: Smooth reveal animations on scroll
- **Bilingual Support**: Czech (default) and English
- **Responsive Design**: Mobile-first approach
- **Conversion Optimized**: Clear CTAs and trust signals

## Sections

1. Hero - Lead capture form (URL + Email)
2. Trusted By - Client logo marquee
3. Problem/Agitation - Pain points
4. Our Approach - 3-phase methodology
5. Services - Full service portfolio
6. Case Studies - Regional government results
7. Testimonials - Google Reviews from officials
8. RAGus.ai - Product showcase
9. Tech Stack - Technologies used
10. Voiceflow Partner - Certification badge
11. FAQ - Common questions
12. About - CEO and company info
13. Final CTA - Calendly integration
14. Footer

## Technologies

- Tailwind CSS (CDN)
- Iconify Icons
- Calendly Widget
- Vanilla JavaScript

## Deployment

Simply serve the `index.html` file. All assets are included locally.
